# Shobhit Kumar

Static personal CV website with recruiter-friendly sections:

- Me
- Work Experience
- Education
- Skills
- Projects

## Local Preview

```bash
cd /Users/shobhitkumar/IdeaProjects/shobhit-cv-site
python3 -m http.server 5173
```

Then open:

```text
http://localhost:5173
```


## Deploy To Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Keep the default static deployment settings.
4. Set the project name to something clean, for example `shobhit-kumar`.

If the Vercel project name is available, the free URL should be:

```text
https://shobhitkumar.vercel.app
```

For `shobhitkumar.dev`, you need to buy the domain and connect it in Vercel domain settings.

## Deploy To Netlify

Drag this folder into Netlify Drop or connect the GitHub repository to Netlify.
The `_redirects` and `netlify.toml` files keep direct tab URLs working.
